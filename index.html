Uncommon Core

At UncommonCore, we used machine learning networks to create and grade schoolwork packets for math students at the elementary school level, all while delivering a clean and easy interface for parents and teachers to keep track of their kids. As part of this project, I developed many general tools for MySQL database interfacing in both Python and PHP, as well as created a library of modular automation tools for web development.

Square1

Square1 is the first-ever PT modality that acts as a complete system for the neuromechanical system of the human body. Our proprietary method assesses each client's body in real time; a simple interview to identify what the body needs at that exact moment, then feeds the appropriate neuronal intervention to correct it. To learn more please get in touch with us on our website below!

Greenstar Group, Inc.

Co-Founder, COO, and Software Developer of Greenstar Group, Inc. Greenstar Group is a financial software company that builds customized software tools for private wealth managers. We utilized a Flask web framework to deliver product that automatically scraped financial data from Yahoo Finance, ran it through a quadratic programmer in python, ported to a variety of MplD3 graphs, and rendered results into a LaTeX template for professional use. Resolved cross-machine dependencies via Docker and hosted containers on AWS.
